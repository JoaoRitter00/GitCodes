Basicamente com programação se pode fazer qualquer coisa, aprendemos hoje a linguagem de programação JavaScript (JS)
Alert
mostra determinada mensagem na maioria das vezes por variaveis
Var
declara uma variavel (normalmente se usa var nome = (função) (valor))
Let
ele só pode trabalhar dentro do escopo dele mesmo
Const
ele é restrito à ser mudado uma vez, para assegurar a integridade dos dados da variante
Prompt
armazena dados do usuario
Number
usa os dados em forma de numero
